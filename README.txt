Это игра танчики, подобная игре на денди.
В ней реализованы:
* 5 уникальных уровней
* Объект управляемый игроком (зеленый танк)
* Объект снаряда
* По мере прохождения сложность уровня увеличивается
* Жизни объекта
* На уровнях пристутствует аптечка, востанавливающая жизнь
* В игре есть начальное меню
